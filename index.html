去除应变温度效应

  • 算法名称：RemoveTemperatureEffectOfStrainFiber
  • 算法功能：适用于剔除应变光纤的温度效应。
  • 算法背景：应变光纤光栅在测量过程中，光纤自身也会受到温度变化影响，因此实际测量结果包括被测物体表面应变和光纤自己的温度应变，需要根据光纤光栅的温度效应系数剔除温度应变。
  • 算法参数
    • 输入与输出
      input 一组应变序列数据，以及一组或多组温度序列数据
      output 修正后的应变序列，修正过程中应变与温度的相关性序列
    • window_size 滑动窗口宽度
      默认为7d，用于计算一定时间内应变与温度的相关性。
    • window_size_min 最小时间范围
      默认为1d，如果滑动窗口内应变实际监测天数不足1天，不予计算。
    • k 温度效应系数
      默认为8.45微应变/℃。
      $$ E = mc^2 $$
    • t 基准温度
      默认为20℃。
    • mode 计算模式
      一组输入温度 mode不生效 直接将这一组温度视作测点附近温度；
      多组输入温度 mode = 0 将输入温度的均值作为测点附近温度；
      mode = 1 计算应变与温度的相关性系数，将相关性系数最高的一组温度作为测点附近温度；
      mode = 2 计算应变与温度的相关性系数，将相关性系数超过阈值的所有温度的均值作为测点附近温度；如果没有一组相关性超过阈值的温度，则按照mode=1方式计算。
    • coef_gate 相关性阈值
      默认为0.8。
